3.1. Matrices in NumPy#

Matrices are used in several fields of study, including, mathematics, computer science and engineering. In this section we will be introducing matrices and the core matrix operations. We will be doing this using NumPy matrices.

3.1.1. Types of Matrices#

Letâ€™s start from the very beginning, by looking at different types of matrices.

  • A matrix consisting of a single row is called a row matrix.

  • A matrix consisting of a single column is called a column matrix. We have already seen this in Practical 2 where we introduced 1-D NumPy arrays or vectors.

  • A matrix which has \(n\) rows and \(n\) columns, that is, the same number of rows and columns, is called a square matrix.

  • A matrix which has \(n\) rows and \(m\) columns, is called a rectangular matrix.

3.1.3.2. Determinant of a 2 x 2 matrix#

A determinant is a scalar that is calculated from a square matrix as follows:

Given a matrix \(A\):

\[\begin{split} A = \begin{bmatrix} a & b\\ c& d\\ \end{bmatrix} \end{split}\]

The determinant of \(A\) is \(|A|\) where \(|A| = ad-bc\)

Let us try to calculate a determinant of a matrix in the example below.

\[\begin{split} A = \begin{bmatrix} 1 & 4 \\ -1 & -1 \\ \end{bmatrix} \end{split}\]

Using the formula \(|A| = ad-bc\), the determinant of \(A\) can be calculated as:

\[\begin{split} \begin{align} |A| & = (1 \times -1) - (4 \times -1)\\ & = -1 + 4 \\ & = 3 \end{align} \end{split}\]

Using NumPy this is coded as:

a = np.array([[1, 4], [-1, -1]])
det_a = np.linalg.det(a)
print(f"{det_a:.2f}")
